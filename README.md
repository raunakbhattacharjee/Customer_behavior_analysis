# 📊 Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI to uncover valuable business insights from retail transaction data. The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, interactive dashboard creation, and presentation of findings through a report and presentation.

The objective is to demonstrate an end-to-end data analytics pipeline, from raw data to actionable insights.

---

## 📂 Dataset

The dataset contains **3,900 customer purchase records** with information such as:

* Customer demographics
* Product category
* Purchase amount
* Payment method
* Shipping type
* Subscription status
* Discount usage
* Customer reviews
* Purchase frequency

## 📋 Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Performed an initial inspection of the data.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicates (if any).
* Corrected data types.
* Prepared the dataset for analysis.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Explored purchasing behavior.
* Identified spending trends.
* Compared customer segments.
* Created visualizations to support insights.

### 4. SQL Analysis

* Imported the cleaned dataset into a relational database.
* Executed SQL queries to answer business questions.
* Performed aggregations, filtering, joins, grouping, and ranking.
* Generated KPI-based insights for decision-making.

### 5. Power BI Dashboard

Developed an interactive dashboard featuring:

* Revenue analysis
* Customer segmentation
* Subscription analysis
* Product performance
* Discount impact
* Shipping preferences
* Interactive filters and KPIs

### 6. Reporting

Prepared a concise business report summarizing:

* Key findings
* Business insights
* Recommendations
* Data-driven conclusions

### 7. Presentation

Created a professional presentation using **Gamma AI**, then refined and exported it as a PowerPoint for effective communication of project outcomes.

## 🚀 How to Run

1. Clone the repository.

   ```bash
   git clone https://github.com/your-username/customer-shopping-behavior-analysis.git
   ```

2. Install the required Python libraries.

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Open the Jupyter Notebook and run the analysis.

4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.

5. Execute the SQL queries provided in the project.

6. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
